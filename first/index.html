普通对象，函数对象

函数对象有三种形式

函数对象都有个prototype,且为普通对象（特例：Function.prototype是函数对象）；

看是普通对象还是函数对象，只要看下就知道了，如果是函数对象则为 function(){}
普通对象则为 Object{普通对象里面包含constructor和proto};

constructor表示是由上面创建的
比如 Object.constructor 为 function Function() { [native code] }
Function.constructor 为 function Function() { [native code] }

Function 与 function 的区别
Function是一个构造器，function是Function实例化的一个对象，只是一个关键字，包含[[call]][[code]]等等属性，可以被执行。
具体使用方法：
function callAnotherFunc(fnFunction, vArgument) {
fnFunction(vArgument);
}
var doAdd = new Function(“iNum”, “alert(iNum + 10)”);
callAnotherFunc(doAdd, 10); //输出 “20”
定义普通函数就用function：

Function-Javascript
Function 对象是 JS 的内置对象之一, 也就是所有函数的构造器. 至于大小写问题, 平时声明函数时所写的 “function xxx() {}” 中的 “function” 只是个关键字. 你可以看一下这个: Function - JavaScript , 注意第一句话: “Function 构造器创建了一个新的 Function 对象. 在 JS 中每一个函数实际上都是一个 Function 对象.”

有两点需要注意：

  1. 注意Object.constructor===Function；//true 本身Object就是Function函数构造出来的
  2. 如何查找一个对象的constructor，就是在该对象的原型链上寻找碰到的第一个constructor属性所指向的对象

原型对象prototype中都有个预定义的constructor属性，用来引用它的函数对象。这是一种循环引用

总结

  1. 原型和原型链是JS实现继承的一种模型。
  2. 原型链的形成是真正是靠proto 而非prototype
